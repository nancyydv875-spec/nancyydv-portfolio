The story behind the code

From a Computer Science classroom in Uttar Pradesh to backend systems running in production — and back to the classroom, this time as the one teaching.

summary

Backend engineer, AI practitioner, trainer

I'm Kaushindar Chaudhary, a software engineer who's spent the last three years building backend systems for e-commerce and ride-hailing products, primarily with Java and Spring Boot. I graduated with a B.Tech in Computer Science Engineering from Dr. A.P.J Abdul Kalam Technical University in 2023, and I'm currently pursuing an M.Tech in Artificial Intelligence at NIET, expected in 2028.

Alongside engineering work, I train students in Java, Python, SQL, web fundamentals, and applied machine learning — something I've done for over two years and for more than a thousand students. I care about writing backend code that holds up under real traffic, and explaining it in a way that someone new to the field can actually follow.
